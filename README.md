# APARTMENT-MANAGEMENT-System
This is a web-based responsive application that can help to manage your multiple Apartments,  Hostel,  flat, building or any kind of real estate property.

It can handle calculations and reporting with all flexible settings, smooth operation, user friendly interface and front-end, full of functionality and compatible features.

This Fully Responsive Software is built with PHP-Codeigniter Framework, Mysql, Javascript, jQuery and Ajax.



To deploy this application, Simply:

   1- Download or clone the repo

   2 - Extract the zip file to your web directory or localhost/htdocs folder if you are on a local server

    3  - Create a database

    4 - Import the SQL file into your database

    5 - open the "database.php" file  located in “harbmvc/config/” Folder

   
    6 - change the following values to mahtch your database settings: 
            
              'hostname' => 'your database host name', 

              'username' => 'your database username',

              'password' => 'your database password',

               'database' => 'Your database name',

   7 -  open your web browser and navigate to your site or e.g http://localhost/your site/ or http://www.yoursite.com/

    8- use the following login Credentials to login as Super Admin:

              Usernam:   admin@admin.com 
              Password:  password
